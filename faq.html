Common Inquiries

Instant answers to the most frequent questions from school administrators, teachers, and accountants.

🚀 Getting Started

The default temporary password for all new staff accounts is 123456. Upon first login, the system will force the user to set a new password before accessing any module.
Follow this order for optimal setup:
1. Settings → School Profile — Set school name, logo, and address
2. Settings → Classes — Create all class groups
3. Staff Module — Register all teachers and staff
4. Students Module — Enroll your students
5. Settings → Fee Structure — Define fee heads
Refer to the Getting Started guide for full details.
Nexus SMS works best on Google Chrome (latest version). It also supports Firefox, Edge, and Safari. Since the app runs on Google Apps Script, Chrome provides the most reliable experience. Ensure JavaScript is enabled and pop-ups are allowed for the app domain.

🎓 Students & Attendance

Yes. Open the student's record via the Actions → Edit menu in the Student Registry. You can change any field including class assignment, parent details, and contact information. Changes apply immediately.
Setting a student to Inactive preserves their record but removes them from:
• Daily attendance rosters
• Fee billing cycles
• Dashboard KPI metrics
You can reactivate them at any time to restore full operations.
Nexus SMS has a double-save guard that prevents duplicate attendance records. If attendance has already been saved for a class on a given date, you'll be alerted. Use the modify feature to correct entries for that date instead of creating new ones.
A 0% attendance on the Today filter means attendance has not been marked yet for that class today. This is expected early in the day before teachers complete their morning marking. Once attendance is saved, the chart updates automatically.

🔐 Security & Access

Sessions are valid for 8 hours from the time of login. After expiration, you'll be redirected to the login screen. Sessions work across multiple browser tabs and persist even if you close and reopen the browser.
As an Admin, go to Staff Module → Actions → Reset Password for that staff member. This sets their password back to the temporary value 123456 and flags their account for a mandatory password change on next login.
There are no software-imposed limits. Nexus SMS is built on Google's cloud infrastructure and is optimized for institutions ranging from 100 to 10,000+ students. Performance scales with your Google Sheets quota.

🔔 SMS & Notifications

Check the following:
1. SMS provider credentials are correctly configured in Settings
2. Parent phone numbers are in the correct international format (e.g., +92 for Pakistan)
3. Your SMS credit balance with the provider is sufficient
4. The SMS feature is enabled in your notification settings
