System Configuration

Tailor every aspect of Nexus SMS to your institution. School profile, theme customization, academic config, class management, fee structures, and SMS settings.

Phase 1: Identity

School Profile

Define your institution's identity. Upload your school logo and set the official school name, address, phone number, and email address.

Global Sync Updated branding is instantly applied across the entire app — sidebar header, PDF receipts, report cards, and SMS signatures.
[Image: settings_school_profile.png]
Phase 2: Appearance

Theme Customization

Customize the look and feel of Nexus SMS. Set your primary accent color and choose between Dark Mode and Light Mode.

  • Primary Color: Affects buttons, links, chart colors, and sidebar highlights
  • Theme Mode: Dark or Light — applied globally for all users
  • Instant Preview: Changes are applied immediately after saving
[Image: settings_theme_config.png]
Phase 3: Academic

Academic Configuration

Set the foundational parameters for your school's operations:

  • Academic Year: Define the current session (e.g., 2025–2026)
  • Timezone: Set your institution's timezone for accurate timestamps
  • Currency: Define the currency symbol for fee displays (Rs, $, £, etc.)

Phase 4: Classes

Class Management

Create and manage your class groups. Each class represents a distinct section (e.g., Grade-01 A, Grade-01 B, Grade-02). Classes appear in all dropdowns across the system.

  • Add Class: Create a new class group with name and optional section
  • Edit: Rename existing classes
  • Delete: Remove classes (only if no students are assigned)
[Image: settings_class_management.png]
Phase 5: Calendar

Holiday Calendar

Define institutional holidays to ensure accurate attendance calculations. Marked holidays are excluded from attendance tracking — students won't be marked absent on holidays.

Attendance Impact When calculating attendance percentages, holiday days are subtracted from the total working days, ensuring accurate reporting.
[Image: settings_holiday_calendar.png]
Phase 6: Finance

Fee Structure Configuration

Define the fee heads and amounts for your institution. Each fee head can have different amounts per class level. This configuration drives the entire Fees module.

Critical Setup Fee structure must be configured before any fee billing can occur. Changes to fee amounts affect future billing only — existing records are preserved.
[Image: settings_fee_structure.png]
Phase 7: Communication

SMS Provider Settings

Configure your SMS gateway provider for automated messaging. Enter the provider API credentials, sender name, and test the connection.

  • Provider API: Enter your SMS gateway API key and credentials
  • Sender Name: The school name that appears as the SMS sender
  • Phone Format: Ensure parent numbers include the country code
[Image: notifications_sms_config.png]
