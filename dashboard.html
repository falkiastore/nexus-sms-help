What is the Dashboard?

The Dashboard is the first page you see after logging into Nexus SMS. It provides a high-level summary of your school's current status, including student counts, staff presence, and financial highlights.

📈

Analytics Cards

Top-level counters for Total Students, Active Staff, Today's Attendance, and Monthly Revenue.

🔔

Recent Alerts

Stay updated with the latest system notifications and automated alerts for fees and presence.

📅

Academic Calendar

Quick view of upcoming holidays, exams, and school events.

How to use the Dashboard

1

Review Key Metrics

Upon login, check the top row of cards. These metrics are updated in real-time based on the latest data from the Students, Staff, and Fees modules.

[Screenshot: Dashboard Metric Cards]
2

Navigate to Modules

Use the sidebar navigation to jump into specific modules. The Dashboard acts as a central hub for all major operations.

[Screenshot: Navigation Sidebar]
3

Monitor Feedbacks

The dashboard provides immediate feedback on system actions. If a student is marked absent or a payment is late, you'll see highlights here.
