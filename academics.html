Academics & Grading Mastery

From exam scheduling to automated report cards—master your school's academic cycle.

Phase 1: Session Setup

Defining Academic Sessions

Start each academic year by defining your Session (e.g., 2026-27). This acts as a container for all student records, attendance, and grades. Ensure you assign Classes to the session to enable enrollment.

[UI Preview: Academic Session Setup]
Phase 2: Examination

Exam Types & Subject Weights

Define your grading criteria for the term. You can set up Monthly Tests, Mid-Terms, and Finals. For each exam, specify the subjects involved and the 'Max Marks'. You can also assign weightages if you use a cumulative grading system.

Phase 3: Evaluation

Rapid Marks Entry

Navigate to Academics > Mark Entry. Select the Class, Exam, and Subject. Nexus SMS provides a spreadsheet-like grid that allows teachers to record marks obtained, teacher remarks, and attendance for the exam period in a single streamlined session.

[Grid Detail: Mark Entry Spreadsheet Interface]
Phase 4: Publication

Automated Report Card Generation

Once marks are verified, the system can generate comprehensive PDF report cards. These include subject-wise grades, percentage calculations, class positions, and attendance summaries automatically plotted into a professional layout.

[Report Sample: Premium Student Result Card]

POWER USER: GRADING SCALES

You can customize your GPA or Grade thresholds (A, B, C...) in the System Settings. Once updated, the system will re-calculate all existing result cards to match the new scale instantly.
