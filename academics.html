Overview

The Academics module allows you to define exam types, manage grading systems, enter marks for students, and generate comprehensive report cards.

📝

Exam Types

Set up Monthly Tests, Mid-Terms, and Annual Exams per academic session.

📊

Mark Entry

Enter and review marks for various subjects across different classes.

🏆

Report Cards

Automatically calculate positions and generate printable report cards for students.

Grading Workflow

1

Create Exam

Go to Academics > Exam Setup. Define the exam name, date, and subjects involved for each class.

2

Input Marks

Select the Class, Exam, and Subject. The list of students will appear, allowing you to enter marks obtained along with any teacher comments.

[Screenshot: Marks Entry Grid]
3

Publish Results

Once marks are verified, you can generate PDF report cards for the entire class or individual students.
