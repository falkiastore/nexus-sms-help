Academic Rigor

Total oversight of the educational cycle. From creating exams and entering marks to generating premium student report cards.

Phase 1: Setup

Exam Creation

Start by creating exam types for each class. Navigate to the Grades module and click Create Exam. Select the class and define the exam type (Mid-Term, Final, Quiz, etc.).

  • Duplicate Prevention: The system prevents creating duplicate exam types for the same class
  • Date Tracking: Each exam records its creation date for audit trails
  • Dependency Check: Exams with graded records cannot be deleted
[Image: academics_create_exam.png]
Phase 2: Management

Exam Table & Search

All created exams appear in a sortable table. Use the search bar to find specific exams. Each row shows the exam type, class, date, and action options.

Quick Actions Click "Enter Marks" directly from the exam table to jump into the marks entry grid for that specific exam and class.
[Image: academics_exam_list.png]
Phase 3: Evaluation

Rapid Mark Entry

The Nexus Marks Grid provides a lightning-fast interface for teachers to input scores. Select the exam for a class to load all students, then enter marks per subject.

  • Max Marks Validation: Prevents entering marks higher than the defined maximum
  • Auto-Tabbing: Tab through fields for rapid data entry
  • Status Tracking: Instantly see which subjects are still pending evaluation
[Image: academics_marks_entry.png]
Phase 4: Configuration

Grade Scale Customization

Customize your institution's grading scale in Settings. Define what percentage ranges correspond to letter grades (A+, A, B, etc.).

Academic Tip Adjust your grading scale to match local educational standards. Common configurations: A+ for 95%+, A for 85%+, B for 75%+, and so on.

📊 Grade Scale

A85% – 94%
B75% – 84%

FBelow 50%
Phase 5: Publication

The Smart Report Card

Generate high-definition PDF report cards that aggregate attendance, subject grades, and teacher commentary into a beautiful, school-branded layout ready for printing.

Publication Note Student results can be locked after publication to prevent unauthorized grade tampering post-distribution.
[Image: academics_report_card.png]
Phase 6: Intelligence

Dashboard Grade Insights

Grade data automatically flows into the Admin Dashboard's Grade Insights Panel, providing real-time visibility into academic performance:

  • 🏆 Top Students: Students ranked by highest overall percentage
  • ⚠️ Weak Students: Students falling below the passing threshold
  • 📉 Weak Subjects: Subjects with the lowest class-wide averages
[Image: dashboard_grade_insights.png]
