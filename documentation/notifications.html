Notification Engine

Automated communication with parents and staff. SMS alerts for attendance, fee reminders, and broadcast messaging — all from within Nexus SMS.

Overview

Notification Center

The Dashboard includes a Recent Notifications widget showing the latest alerts sent by the system. Administrators can monitor SMS dispatch statuses and read/unread states.

  • Real-Time Feed: New notifications appear instantly after triggering events
  • Mark as Read: Click any notification to mark it as read
  • History: Full notification log available in the Notifications module
[Image: dashboard_notifications_widget.png]
Setup

SMS Provider Configuration

Configure your SMS gateway in Settings → SMS Configuration. Nexus SMS integrates with SMS providers to send text messages directly to parent phone numbers.

Configuration Required SMS will not be sent until you complete the provider setup. Ensure phone numbers are provided in the correct international format (e.g., +92 for Pakistan).
[Image: notifications_sms_config.png]
Automated

Attendance SMS Alerts

When a teacher saves attendance, the Notification Engine automatically sends SMS alerts to parents of students marked as 'Absent'. The message includes the student's name and school information.

Message Template Default: "Dear Parent, [StudentName] was marked absent today from [SchoolName]." — Customize this template in Settings → SMS Templates.
📱

SMS Alert Flow

1. Teacher marks attendance & saves
2. System identifies Absent students
3. SMS sent to parent phone numbers
4. Notification logged to dashboard
Financial

Fee Reminder Notifications

Send overdue payment reminders to parents with outstanding balances. These can be triggered manually by the administrator or accountant from the Fees module.

  • Individual Reminder: Send to a specific student's parent
  • Bulk Reminder: Send to all students with overdue fees in a class
  • Custom Message: Customize the reminder text before sending
[Image: notifications_center.png]
Broadcast

Bulk Broadcast Messaging

Send announcements to entire class groups or the entire school at once. Compose your message, select the target audience, and dispatch.

Usage Tip Use bulk broadcasts for school announcements, holiday notifications, parent-teacher meeting reminders, and emergency alerts.

📣 Broadcast Targets

🏫 Entire SchoolAll parent contacts
📚 Specific ClassClass-level parents
👤 IndividualSingle parent
