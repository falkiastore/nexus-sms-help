Common Inquiries

How do I reset a Teacher's password?

Administrators can reset any staff password via the Staff Management module. Go to Staff -> Action -> Reset Password.

Can I export financial data to Excel?

Absolutely. The Reporting & Export module allows you to pull clean datasets for all financial transactions in standard formats.
