Reports & Data Export

Download, analyze, and share your school's data. Export student records, fee transactions, and attendance reports — all in standard formats.

Overview

The Export Module

The Export module provides a centralized interface for downloading data from any major section of Nexus SMS. Available to Admin and Accountant roles.

  • Format: Data is exported as CSV/Excel-compatible spreadsheets
  • Access Control: Export visibility is role-restricted — Accountants can only export financial data
  • Direct Download: Files download instantly to your browser
[Image: export_module_view.png]
Students

Student Data Export

Download the complete student registry including all profile fields — names, class assignments, parent contacts, addresses, and enrollment status. Use the class filter to export a single class group.

Bulk Operations Exported student data can be used for mass communications, school directory printing, or data migration to other systems.

Financial

Fee Transaction Export

Export all fee payment records with timestamps, amounts, payment methods, and student details. Filter by date range or class for targeted exports.

  • Transaction ID: Unique reference for every payment
  • Date Filters: Export records from specific month or custom date range
  • Reconciliation: Use exported data for accounting reconciliation with bank statements

Attendance

Attendance Reports

Generate monthly attendance summaries by class. Each report shows daily attendance records with percentage calculations for each student.

Report Use Monthly attendance reports are commonly required for regulatory compliance and can be shared with parents during parent-teacher meetings.
[Image: dashboard_attendance_chart.png]
Academic

Report Card Generation

Generate branded PDF report cards from the Academics module. These include subject-wise marks, grade calculations, attendance summaries, and teacher remarks — all formatted with your school's logo and colors.

Prerequisite Report cards require complete marks entry for the selected exam type. Ensure all subjects have been graded before generating.
